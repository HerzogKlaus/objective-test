# Объектив - тестовое задание



## Запуск

Сборка в режиме dev:

```bash
npm run build:dev
```

Сборка в режиме prod:

```bash
npm run build:prod
```

Запуск dev-server:

```bash
npm run serve
```

## Переменные окружения

Использовал только при создании конфига webpack следующие переменные окружения:

**mode** - режим сборки

**port** - порт на котором стартует dev-server
